# Utilities

The `nlvib.utils` package provides two sub-modules:

- `nlvib.utils.transforms` — FFT-based transforms and the Alternating Frequency-Time (AFT)
  pipeline (Krack & Gross (2019) §2.3)
- `nlvib.utils.linalg` — numerical utilities for continuation (scaling, arc length)
