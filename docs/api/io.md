# IO

The `nlvib.io` module provides readers and writers for CalculiX FRD/mesh file formats,
enabling finite-element model import and animation export.

!!! note
    Full CalculiX CGX compatibility requires an external CalculiX installation.
    The IO functions are tested with synthetic mesh files in the test suite.
